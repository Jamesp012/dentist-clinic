## 🚀 Quick Start (2 minutes)

1. Open **Dental Charting** module
2. **Select a patient** from dropdown
3. **Click some teeth** to mark conditions (they change color)
4. **Click [Add New Chart]** blue button at the bottom
5. Chart is saved! View in "Chart History" below
6. **Try again** - mark different teeth
7. **Click [Add New Chart]** again
8. **Expand** a chart to see details
9. **Click [Load Chart]** to reload it

**That's it!** You're using chart history.

## 🔍 Key Concepts Explained

### Chart History
Complete record of all dental charts created for a patient, stored with timestamps.

### Add New Chart
Button that saves current tooth conditions as a historical entry and clears for new chart.

### Load Chart
Function to restore a previous chart into the editor for modification or comparison.

### Timestamp
Automatic date and time recording when a chart is saved.

### Tooth Conditions
The markings on each tooth (cavity, filling, crown, etc.) saved in the chart.

### Chart Expansion
Clicking a chart to see full details including conditions and notes.
