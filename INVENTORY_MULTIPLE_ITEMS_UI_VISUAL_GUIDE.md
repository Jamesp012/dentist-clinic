## Animation & Interactions

### Adding an Item
```
1. User selects item from dropdown
   Dropdown highlights selection

2. User enters quantity
   Input field validates (min: 1)

3. User clicks "Add Item"
   Button shows loading state: [Adding...]

4. Item appears with animation
   Slides in from top
   Green checkmark shows
   вң“ Item added

5. Form resets for next item
   Dropdown back to "Select..."
   Quantity back to 1
   Ready to add more
```

### Removing an Item
```
1. User hovers over item card
   Card gets light background
   Remove button (вң•) becomes visible

2. User clicks remove button
   Item fades out
   Removed from list
   Count updates: "(3 items)" вҶ’ "(2 items)"
```
